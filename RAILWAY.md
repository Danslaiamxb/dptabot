# Деплой на Railway — без VPN, без командной строки на своём компе

Railway хостит бота на своих серверах, поэтому проблемы с блокировкой
Telegram у тебя на компе больше не будет — бот работает не у тебя, а в облаке.

## Что нужно

- Аккаунт GitHub (бесплатно, если нет — регистрируешься на github.com)
- Аккаунт Railway (регистрируешься на railway.com через GitHub — один клик)
- Токен бота от @BotFather (тот же что уже есть)

## Шаги

### 1. Залей код на GitHub

1. Зайди на github.com, нажми **New repository** (зелёная кнопка)
2. Назови репозиторий как хочешь (например `dota-bot`), сделай **Private**
   (чтобы токен и настройки не светились публично)
3. Жми **Create repository**
4. На следующей странице — жми **uploading an existing file**
5. Перетащи туда файлы: `bot.py`, `requirements.txt`, `Procfile`
   (config.json и .bat файлы НЕ нужны)
6. Внизу жми **Commit changes**

### 2. Подключи Railway

1. Зайди на railway.com, залогинься через GitHub
2. **New Project** → **Deploy from GitHub repo**
3. Выбери свой репозиторий `dota-bot`
4. Railway сам увидит `requirements.txt` и `Procfile`, начнёт разворачивать

### 3. Впиши токен и Steam ID

1. В проекте на Railway зайди в свой сервис → вкладка **Variables**
2. Добавь переменную: `BOT_TOKEN` = твой токен от BotFather
3. Добавь переменную: `STEAM_ID32` = `1125119990`
4. Сохрани — Railway автоматически перезапустит бота с новыми настройками

### 4. Проверь что работает

1. Вкладка **Deployments** → открой последний деплой → **View Logs**
2. Должна быть строка `Бот запущен со Steam ID..., жду сообщений...`
3. Иди в Telegram, пиши своему боту `/start` — должен ответить

## Важно про бесплатный тариф

Railway даёт бесплатные кредиты (обычно $5 разово + понемногу каждый месяц) —
для одного бота на постоянной работе этого достаточно на долгое время.
Если кредиты закончатся, бот на время остановится, пока не пополнишь или
не дождёшься следующего месяца.

## Если бот не отвечает после деплоя

Смотри Logs (шаг 4) — там будет видна ошибка текстом. Скинь мне текст
ошибки, разберём.
